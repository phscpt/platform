Frameworks

Built primarily with Flask, along with Water.css for frontend styling, Markdown for problem creation, and MathJax for problem rendering. Deployed on PythonAnywhere.

Source code: https://github.com/phscpt/platform

Environment

All commands are run on a server with Ubuntu 20.04.2 LTS.

  • Python 3.9.5: python3 [filename].py
  • Python 2.7.18: python2.7 [filename].py
  • Java: javac [filename].java; java [filename] (openjdk 11.0.11)
  • C++: g++ [filename].cpp; ./a.out (C++14, g++ version 9.3.0)
